# C Programming language

C is a general purpose programming language developed by Dennis Ritchie at Bell Labs between 1972 to 1973.

Dennis Ritchie is a computer scientist at Bell Labs who create C language, making it fast and efficient for system programming.

Ken Thompson: Collaborated with Ritchie on UNIX and the B language, providing the foundation upon which C was built.

Brain Kernighan: He was co-authored for the first book on the language. "The C Programming Language", with Ritchie, making him instrumental in its populaization.

Ritchie and Thompson received the National Medal of Technology in 1999 for their contribution.

# History of C Programming lanauge

- The C Programming language was created in year 1972 by **"Dennis Ritchie"** at AT&T Bell Laboratories.
- It was originally designed to build the UNIX operating system. providing a way to write system software that was more portable and easier to maintain than assembly language.

## Origins and Early Evolution

C did not emerge in isolation; it was the result of a progression of languages designed for efficiency and system control.

- **ALGOL (1960)**: The root of many modern programming language, it introduced the concept of structured programming.
- **BCPL (1967)***: Developed by Martin Richards, it was a typeless language used primary for writing compilers.
- **B Language (1970)**: Created by Ken Thompson, B was a simplified version of BPCL used i early UNIX utilities but lacked advanced datatypes.
- **Birth of C(1972)**: Ritchie enhanced the B language by adding datatypes ( like int, char and float0 and structures. By 1973, the UNIX kernel was entirely re-written in C.
